## Tools

### credential_scanner.py
Automated detection of credential vulnerabilities in agent workspaces.

**Finds:**
- Hardcoded credentials (API keys, tokens, private keys)
- Insecure file permissions
- Credentials in Git history
- 7+ credential types (OpenAI, Anthropic, AWS, Solana, JWT, etc.)

**Usage:**
```bash
wget https://github.com/therealsparky1/sparky-sentinel/raw/main/credential_scanner.py
python3 credential_scanner.py /path/to/workspace
```

**Features:**
- Educational mode (explains WHY issues matter)
- Zero dependencies (stdlib only)
- Redacted output (safe to share)
- Platform agnostic (Linux/Mac/Windows)
